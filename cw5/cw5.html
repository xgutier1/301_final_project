High-Level Languages

When we thing about computer programmers, we are probably thinking about people who write in high-level programming languages.

High-level languages are written in a form that is close to our human language, enabling to programmer to just focus on the problem being solved.

Examples include: C, C++, Java, Pascal, Python, Visual Basic.

Advantages

  1. Easier to modify as it uses English like statements
  2. Easier/faster to write code as it uses English like statements
  3. Easier to debug during development due to English like statements
  4. Portable code – not designed to run on just one type of machine

My Favorite Language

Glossary

HTML
Hypertext Markup Language, or HTML, is the standard markup language for web pages. With HTML you can create your own Website. HTML is easy to learn - You will enjoy it!
CSS
Cascading Style sheets, or CSS, is a language that describes the style of an HTML document. CSS describes how HTML elements should be displayed.
AJAX
Asynchronous JavaScript and XML, or AJAX, is a developer's dream, because you can read data from a web server after the page has loaded, update a webpage without reloading the page, send data to a web server, etc. all in the background.
JSON
Javascript Object Notation, or JSON, is a syntax for storing and exchanging data. JSON is text, written with JavaScript object notation.
